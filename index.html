Get Started

1. Share Button

First you have to find a share button for the website. It is different and it may look different on your device and os. But at all your browser should also have a share function or other function for the following steps. If you do not find it you can google for a solution for adding Websites to homescreen.

2. Add to Homescreen

After that you have to find the option to add this website to your homescreen. This adds the website like an app to the homescreen. It does not have any acess to your local storage like any other app because it is still running on the web. You also can use this web application in your browser but it just has benefits to add it to your homescreen.

3. Last Steps

Finally you have to name your app however you want it. The default name for this website is VG Trainer. You also can see the url it leads to which be loaded in your web app at the end. This is the last step of adding this website as an app to your personal device. Good luck with learning Latin for now and focus on learning it!
